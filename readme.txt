## Languages and Technologies Used

- **Frontend:** HTML, CSS, JavaScript, Tailwind CSS  
- **Backend:** PHP  
- **Database:** MySQL  
- **Tools:** VSCode, XAMPP  

---

*-----------------------------*
* Running inside the VSCode
*-----------------------------*
1. open a terminal in VSCode
2. on terminal type: php -S localhost:8000
3. open a web browser
4. on web browser url type: https://localhost/ERFEO/landingpage/index.php

*-----------------------------*
* Run on localhost must be in htdocs
*-----------------------------*
1. copy the ERFEO folder to: c:\xampp\htdocs\
2. open web browser
2. on web browser url type: https://localhost/ERFEO/landingpage/index.php
